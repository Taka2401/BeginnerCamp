
# Beginner's Camp
**キャンプを始めたい、初心者向けに特化した情報を提供するアプリです！**

キャンプをしてみたい！でも、キャンプ場の予約や用品を準備するのに時間が掛かりませんか？<br />
**Beginner's Campではその手間を省き**、初心者向けにオススメのキャンプ場等を紹介します。<br />

### テーマを選んだ理由
僕自身、キャンプに行ったことがありませんでした。
キャンプに行こうと考え、キャンプ場を調べるもどういうキャンプ場がいいのか、何を自分で用意すればいいのか分からず、<br />
時間が掛かり結局断念したことがあり、初心者向けに特化したサイトを制作することで問題解決したいと決めました。<br />
既存のキャンプサイトではなく、様々な情報が豊富にあるがゆえに、結局何を選べばいいか分からなくなっていたので、<br />
あらかじめターゲットを初心者に絞り、キャンプに詳しくない人でも行きやすいキャンプ場を紹介することにしました。<br />
また、ユーザーが投稿できるのでサイトからのオススメだけではなく、多角的に情報を得ることができ、<br />
ユーザ間で情報を共有することができます。<br />
それによって、サイトとユーザーが共にキャンプに行きやすい情報を提供できるサービスになれるように制作しました。
